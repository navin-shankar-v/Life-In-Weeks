# Life Calculator

A visual "Your Life in Weeks" calculator built with React + Vite. Enter your birth date to see a grid of every week in a 100-year life—lived weeks, this week, and remaining—so you can visualize how much time you have.

## Features

- **Birth date input** — Enter your birth date to personalize the visualization
- **Interactive grid** — Each dot = 1 week; rows represent years (52 weeks each)
- **Color-coded** — Blue = lived, amber = current week, gray = remaining
- **Hover to see dates** — Hover over any dot to see the corresponding week and date

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173/](http://localhost:5173/) in your browser.

## Build

```bash
npm run build
npm run preview
```

## Tech Stack

- React 19
- Vite 7
